# 🎬 Casa dos Cinéfilos

Uma rede social de cinema onde cinéfilos se encontram, descobrem filmes e compartilham suas opiniões.

---

## 🌐 Acesse o site

**[brunopere02.github.io/casa-dos-cin-filos](https://brunopere02.github.io/casa-dos-cin-filos/)**

---

## ✨ Funcionalidades

### 🎥 Catálogo de Filmes e Séries
- Tendências da semana atualizadas em tempo real
- Filmes e séries mais bem avaliados
- Em Breve nos cinemas
- Filtro por gênero, ano e nota mínima
- Paginação com botão "Carregar Mais"
- Busca de filmes e séries

### 🎬 Página Individual
- Página dedicada para cada filme ou série
- Hero imersivo com banner em tela cheia
- Trailer do YouTube integrado
- Elenco principal com fotos
- Temporadas e episódios (séries)
- Onde assistir no Brasil (Netflix, Prime, etc.)
- Avaliação do público com barra de progresso
- Detalhes técnicos (orçamento, bilheteria, emissora)
- Filmes e séries similares recomendados

### ❤️ Favoritos
- Salvar filmes e séries favoritos
- Acesso rápido pela navbar
- Persistência local no navegador

### 👤 Sistema de Usuários
- Cadastro com e-mail e senha
- Login seguro via Firebase Authentication
- Escolha de avatar personalizado
- Perfil com nome, bio e foto
- Estatísticas de resenhas, seguidores e seguindo
- Seguir outros cinéfilos

### 📰 Feed Social
- Feed global com todas as resenhas
- Feed personalizado de quem você segue
- Publicar resenhas com nota em estrelas
- Reações estilo Facebook (❤️ Amei, 👍 Curti, 😂 Hahaha, 😢 Triste, 👎 Não gostei)
- Comentários com picker de emojis
- Clique no filme da resenha vai direto para a página do filme

---

## 🛠️ Tecnologias

| Tecnologia | Uso |
|---|---|
| HTML, CSS, JavaScript | Frontend completo |
| TMDB API | Dados de filmes e séries |
| YouTube Embed | Trailers integrados |
| Firebase Auth | Login e cadastro de usuários |
| Firebase Firestore | Banco de dados de resenhas e perfis |
| DiceBear API | Avatares personalizados |
| GitHub Pages | Hospedagem gratuita |

## 🚀 Como usar

1. Acesse o site pelo link acima
2. Crie uma conta ou faça login
3. Explore filmes e séries no catálogo
4. Favorite o que quiser assistir
5. Escreva resenhas no Feed
6. Siga outros cinéfilos e interaja com as resenhas deles
