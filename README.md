# 🎬 Casa dos Cinéfilos

Site de catálogo de filmes e séries com avaliações, desenvolvido com HTML, CSS e JavaScript puro integrado à API do TMDB.

## 🌐 Acesse o Site

**[https://brunopere02.github.io/casa-dos-cin-filos/](https://brunopere02.github.io/casa-dos-cin-filos/)**

## 📋 Sobre o Projeto

Site estilo Netflix para descobrir filmes e séries, ver sinopses e avaliar com estrelas.

## 🚀 Tecnologias Utilizadas

- **HTML5**
- **CSS3** (design responsivo estilo Netflix)
- **JavaScript** (consumo de API)
- **TMDB API** (dados de filmes e séries)

## 🎮 Funcionalidades

- ✅ Tendências da semana
- ✅ Filmes populares com filtro por gênero
- ✅ Séries populares com filtro por gênero
- ✅ Em breve nos cinemas
- ✅ Busca de filmes e séries
- ✅ Modal com sinopse e detalhes
- ✅ Sistema de avaliação com estrelas
- ✅ Design escuro estilo Netflix
